# DP-300: Administering Microsoft Azure SQL Solutions
**Data:** 8 de Junho de 2026

Profissionalmente, eu sempre mantive uma cadência em relação a provas de certificação, por diferentes motivos: colocação profissional, comprovar que eu sabia o básico sobre um produto ou serviço, não ser questionado quando entregava meus treinamentos ou prestava consultoria e, mais importante, sempre tentar me manter atualizado.
Meu histórico é público e está disponível: https://learn.microsoft.com/en-us/users/luticm79/transcript/7xomoa8gk3l0kzk

Em 30/05/2026, fiz a prova *DP-300: Administering Microsoft Azure SQL Solutions*, adicionando mais uma certificação ao currículo: *Microsoft Certified: Azure Database Administrator Associate*.

Escolhi essa prova pois voltei a me aproximar do SQL Server, com ênfase em Azure SQL. Então, nada melhor do que colocar uma prova no calendário para estabelecer um prazo. Normalmente, o meu modo de operação é revisar os tópicos da prova e planejar uma rotina de estudo, passando pelos learning paths  (https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/dp-300 e https://learn.microsoft.com/en-us/training/courses/dp-300t00). 

Diferentemente das certificações que fiz no passado, desta vez utilizei outra metodologia: parti dos assessments no portal Microsoft Learn e fui revisando as perguntas que eu errava, bem como aquelas em que a explicação ou uma das respostas continha referência a uma configuração ou tema que eu gostaria de aprofundar.  (https://learn.microsoft.com/en-us/credentials/certifications/azure-database-administrator-associate/practice/assessment?assessment-type=practice&assessmentId=58&practice-assessment-type=certification)

Comparando com a prova, os assessments se mostraram bastante interessantes, cobrindo os tópicos listados, os formatos de pergunta e acelerando significativamente meu tempo de estudo. Também utilizei o simulado da Person (disponível na O'Reilly) [DP-300 Administering Microsoft Azure SQL Solutions](https://learning.oreilly.com/certifications/9780135448687/), e se tivesse que escolher apenas um, ficaria tranquilamente com o do Microsoft Learn. 
Existem estilos de perguntas baseados em cenários que os assessments não cobrem, mas encontrei apenas uma pequena quantidade de questões desse tipo na prova (imagino que isso seja padrão para essa certificação).

Considero essa uma certificação de complexidade média. Ela não exige que o candidato decore sintaxe (o que é excelente, especialmente nos dias de hoje) e acredito que cumpre bem seu objetivo: validar se o candidato possui uma boa base para avaliar diferentes soluções com Azure SQL, cenários híbridos, HA/DR e segurança (no meu caso, muitas questões sobre segurança!).

Refletindo sobre a experiência, foi interessante perceber uma sensação muito positiva: a de concluir uma atividade técnica sem qualquer ajuda de IA. Não me entenda mal, utilizo soluções de IA frequentemente e acho que devemos aprender a ser cada vez mais produtivos, mas tenho dúvidas do impacto no nosso poder cognitivo (tema para um outro post).

Para você que trabalha com banco de dados e Azure, espero que vc tenha a oportunidade de tirar essa certificação.
